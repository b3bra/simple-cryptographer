# Simple Cryptographer

## Installing dependencies
* Windows
    ```sh
    pip install -r requirements.txt
    ```
* Linux
    ```sh
    pip3 install -r requirements.txt
    ```

## Launching the application
* Windows
    ```sh
    set FLASK_APP = main.py
    ```
    ```sh
    flask run
    ```
* Linux
    ```sh
    export FLASK_APP = main.py
    ```
    ```sh
    flask run
    ```
